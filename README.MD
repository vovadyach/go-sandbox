## Quick Start
```bash
git clone https://github.com/vovadyach/go-sandbox
cd go-sandbox
cp .env.example .env
docker compose up db -d
cd api
go mod tidy
go run cmd/migrate/main.go
go run cmd/seed/main.go
go run cmd/server/main.go
```
